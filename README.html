Enderman Fix

エンダーマンの挙動を修正するMODです。

動作環境

  • Minecraft 1.7.10
  • Minecraft Forge 10.13.4.1558

導入方法

シングル プレイ

  1. JARファイルをダウンロードします。
  2. JARファイルをmodsフォルダーに配置します。
  3. Minecraftを起動し、エンダーマンの挙動などが変わっていれば導入に成功しています。

マルチ プレイ

シングル プレイと同じですが、クライアント側で導入していなくてもサーバー側で導入していれば反映されます。

修正される項目

エンダーマン

デスポーン条件の変更

ブロックを持っている時はデスポーンしないようになります。
また、ピースフルに変更されるなどして強制的にデスポーンさせられた場合はブロックをその場にドロップします。

ドロップ処理の追加

ブロックを持っている時に倒された場合、ブロックをドロップするようになります。

エンドポータル

ポータルブロックを地上世界(Dimension ID=0)以外で設置した後に消滅する仕様を修正します。
